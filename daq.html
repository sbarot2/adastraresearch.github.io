CalBox Module

The CalBox is a multi-sensor, high-isolation DAQ module designed for thermal, electrical, and calorimetric measurements.

  • PT1000 RTD inputs via custom shielded USB cables
  • K-type thermocouple inputs
  • Isolated high-voltage tolerant ADC channels
  • Pump/flow control and power measurement
  • Ethernet communication

RedBox Module

  • 12V and 24V DC outputs
  • Multiple current shunt channels
  • Digital IO with selectable logic levels
  • RS-232 & RS-485 interfaces
  • AC mains control relay

Data Storage

Data is stored in a structured SQLite schema with experiment/session tracking, channel groups, time-series values, logs, and metadata.
